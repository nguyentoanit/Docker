## A few commom commands
- List cron file:

```
crontab -l
```
- Edit cron file:

```
crontab -e
```
- Start/Stop/Restart apache:

```
service httpd start/stop/restart
```
- Start/Stop/Restart MySQL:

```
service postgresql start/stop/restart
```
