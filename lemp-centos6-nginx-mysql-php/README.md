## A few commom commands
- List cron file:

```
crontab -l
```
- Edit cron file:

```
crontab -e
```
- Start/Stop/Restart apache:

```
service nginx start/stop/restart
```
- Start/Stop/Restart MariaDB:

```
service mysql start/stop/restart
```
